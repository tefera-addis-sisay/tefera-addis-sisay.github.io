This work investigates security monitoring, anomaly detection, and attack analysis in VES-based O-RAN systems.

The research explores:

- VES event monitoring
- Real-time anomaly detection
- Network attack analysis
- Intelligent security monitoring
- AI-assisted threat detection
- Communication system security

The work is currently unpublished and under continuous development as part of ongoing research activities.
