<h2>VES Collector for Network Telemetry</h2>

<p>

  <li>Network telemetry processing</li>
  <li>Event-driven security analytics</li>
</ul>

<h3>🛠 Technologies</h3>
<ul>

  <li>Monitoring systems</li>
</ul>
