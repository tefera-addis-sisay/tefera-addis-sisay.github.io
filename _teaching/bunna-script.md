# Bunna Script (እንጉርጉሮ ስክሪፕት)

**Type:** Pre-University Course

**Location:** Ethiopia  
**Date:** 2025

---

## 📘 Course Description

This course introduces Bunna Script, an Amharic-based educational scripting language designed to help pre-university STEM students learn programming concepts in a simplified and local-language format.

It focuses on building computational thinking skills without requiring prior knowledge of English-based programming syntax.

---

## 🎯 Learning Objectives

By the end of this course, students will be able to:

- Understand basic programming logic using Amharic keywords
- Develop computational thinking and problem-solving skills
- Work with variables, conditions, and loops
- Translate simple problems into algorithmic steps
- Prepare for future programming languages like Python and C++

---

## 🧠 Topics Covered

- Introduction to Bunna Script
- Variables and assignment (እንቅስቃሴ)
- Conditional statements (እንደሆነ / if logic)
- Loops and repetition (መደጋገም)
- Basic functions and structure
- Writing simple algorithms using Amharic syntax

---

## 🧪 Assessment

- Assignments: 30%
- Exercises: 20%
- Midterm test: 20%
- Final project: 30%

---

## 🛠 Teaching Notes

This course emphasizes conceptual understanding of programming using local language support.

Students are encouraged to focus on logic and problem-solving rather than syntax memorization.
