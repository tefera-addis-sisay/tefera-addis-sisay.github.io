## Course Description

**Database Systems (50.043)** introduces the foundations of modern database management systems and large-scale data processing platforms. The course covers relational database concepts, database design, query languages, storage and indexing techniques, transaction management, and query optimization. It also introduces contemporary big data technologies and cloud-based data processing environments.

Students gain hands-on experience designing and implementing database applications using relational database systems, writing advanced SQL queries, and exploring distributed data processing frameworks such as Apache Spark and cloud-based infrastructures.

* Introduction to Database Systems
* Entity-Relationship (ER) Modeling

* Relational Algebra

---

## Learning Outcomes

By the end of the course, students were able to:

* Design and implement relational database applications.
* Develop conceptual and logical database schemas.
* Write complex SQL queries for data retrieval and manipulation.
* Understand database internals, indexing, and storage mechanisms.
* Analyze query execution strategies and performance trade-offs.
* Explain transaction management and concurrency control mechanisms.
* Compare traditional database systems with modern big data platforms.
* Develop and deploy cloud-based data applications.
* Apply distributed data processing techniques using modern frameworks.
