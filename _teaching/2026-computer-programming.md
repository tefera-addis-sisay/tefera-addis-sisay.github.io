## Course Description

**Computer Programming (C/C++)** introduces students to fundamental programming concepts using the C and C++ languages. The course emphasizes structured programming, problem-solving techniques, algorithm development, and the basics of object-oriented programming.

Students learn how to design, implement, test, and debug programs while building strong computational thinking and software development skills.

---

## Topics Covered

- Introduction to Computers and Programming
- Problem Solving and Algorithm Design
- Basics of C Programming
- Variables, Data Types, and Operators
- Control Structures (if, loops, switch)
- Functions and Modular Programming
- Arrays, Strings, and Pointers
- Memory Management Basics
- Introduction to C++
- Object-Oriented Programming Concepts (classes, objects)
- File Handling in C/C++
- Debugging and Program Testing

---

## Learning Outcomes

By the end of the course, students will be able to:

- Understand fundamental programming concepts and logic.
- Write, compile, and debug programs in C and C++.
- Design algorithms to solve computational problems.
- Use functions, arrays, and pointers effectively.
- Apply object-oriented programming principles.
- Develop structured and efficient software solutions.

---

## Teaching Methodology

The course includes:

- Interactive lectures
- Live coding demonstrations
- Weekly programming lab exercises
- Problem-solving assignments
- Group discussions
- Mini programming projects

Students are expected to practice regularly and develop strong coding habits through hands-on exercises.
