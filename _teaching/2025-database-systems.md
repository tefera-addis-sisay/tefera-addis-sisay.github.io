## Course Description

**Database Systems (50.043)** introduces the principles, design, implementation, and management of modern database systems. The course covers conceptual and logical database design, relational data models, SQL, normalization, indexing, query processing, transaction management, concurrency control, and recovery mechanisms. Students are also introduced to modern data-intensive computing platforms and scalable data processing techniques.

The course emphasizes both theoretical foundations and practical implementation through hands-on exercises, programming assignments, and project-based learning.

## Learning Outcomes

By the end of the course, students will be able to:

- Design conceptual, logical, and physical database schemas.
- Develop relational database applications using SQL.
- Apply normalization techniques to improve database quality.
- Analyze and optimize database queries.
- Understand indexing structures and storage mechanisms.
- Explain transaction processing and concurrency control techniques.
- Evaluate database recovery and fault-tolerance mechanisms.
- Design scalable solutions for data-intensive applications.
- Apply modern database concepts in distributed and cloud environments.
- Critically assess trade-offs in database system design.
